# Vehicle Rental System

A full-stack vehicle rental system built with React (Vite) for the frontend and Node.js/Express with MongoDB for the backend. This application allows users to browse, book, and pay for vehicles while giving administrators the ability to manage vehicles, users, and bookings.

---

## Features

### User Features
- User registration and login.
- Browse available vehicles with filters (category, location, price range).
- Book vehicles with start and end dates.
- Make payments for bookings.
- View booking history and status.
- Rate and review completed bookings.

### Admin Features
- Admin login with role-based authentication.
- View all vehicles, users, and bookings in a dashboard.
- Approve or reject vehicle listings.
- Monitor all payment transactions.
- See booking status for each vehicle.
- Manage user accounts (view, role information).
- Logout from any page.

---

## Tech Stack

- Frontend: React 18, Vite, Tailwind CSS, Axios, React Router DOM.
- Backend: Node.js, Express.js, MongoDB, Mongoose, bcrypt for password hashing, JWT for authentication.
- Database: MongoDB
